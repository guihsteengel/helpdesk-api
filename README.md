# HelpDesk API

API de gerenciamento de chamados desenvolvida em C# com .NET.

## 🚀 Tecnologias
- .NET
- Entity Framework Core
- PostgreSQL
- Swagger

## 📌 Funcionalidades
- Criar chamados
- Listar chamados
- Atualizar chamados
- Deletar chamados

## 🗄️ Banco de Dados
- PostgreSQL
- Migrations com Entity Framework

## ▶️ Como rodar o projeto

1. Configure o PostgreSQL
2. Atualize a connection string no appsettings.json
3. Execute:


dotnet run


4. Acesse o Swagger:

https://localhost:xxxx/swagger
